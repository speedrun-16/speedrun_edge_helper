# Speedrun: Edge Helper
![Author](https://img.shields.io/badge/Author-PWNED-8cf?style=for-the-badge "Author") ![Version](https://img.shields.io/badge/Version-1.0-blue?style=for-the-badge "Version")

While crouching on the ground, draws a beam square around the player's crouch footprint when their center hangs over an edge.

![preview](https://github.com/user-attachments/assets/95e868f9-9aeb-4fb5-a8be-575343c883a8)

## ☰ Dependencies
- [visual](https://github.com/speedrun-16/visual)

## ☰ Settings

Toggled per player via `sr_edge_helper_set_enabled`. \
Disabling resets draw state immediately so no leftover outlines remain.
